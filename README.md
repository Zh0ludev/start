# Telegram-бот для снабженца

## Описание проекта

Telegram-бот для автоматизации работы снабженца с базой поставщиков. Позволяет быстро находить, добавлять и управлять информацией о поставщиках через удобный интерфейс мессенджера.

## Пользователи и роли

### Администраторы
- Основные пользователи: 2 человека
- Полный доступ ко всем функциям бота

### Обычные пользователи
Функционал с ограниченными правами (предусмотрен для будущего расширения):
- Просмотр информации о поставщиках
- Поиск по базе
- Добавление комментариев
- Без прав на удаление и приоритизацию

## Основные возможности

### Поиск поставщиков
- Поиск по городам
- Поиск по категориям товаров
- **Комбинированный поиск** - одновременное применение нескольких фильтров
- **Нечеткий поиск** - поиск с учетом возможных опечаток
- **Сортировка результатов**:
  - По имени
  - По приоритету

#### Отображение результатов
- **Краткая карточка** для списка результатов:
  - Название компании
  - Имя поставщика
  - Email (критичное поле)
- **Подробная карточка** - открывается при нажатии на название компании:
  - Все данные поставщика
  - История комментариев
  - Кнопки управления (редактировать, удалить, добавить в избранное, изменить приоритет)

### Информация о поставщиках

Для каждого поставщика хранится и отображается:

#### Контактные данные
- Фамилия
- Имя
- Телефон
- Адрес электронной почты

#### Данные компании
- Название компании
- Физический адрес
- Город
- ИНН
- Сайт компании

#### Коммерческие условия
- Категории товаров
- Минимальная сумма заказа
- Условия оплаты
- Время работы

### Управление базой поставщиков

#### Добавление новых поставщиков
- Внесение информации о новом поставщике
- **Обязательное поле**: только ИНН
- Все остальные поля опциональны
- **Автоматическая проверка на дубликаты по ИНН** перед добавлением

#### Редактирование данных поставщика
- Изменение любых данных существующего поставщика
- Все изменения логируются в историю с сохранением старых значений
- **Критичное поле**: адрес электронной почты (email)

#### Комментарии
- Возможность оставлять заметки о каждом поставщике
- История комментариев

#### Приоритизация
- Установка и изменение приоритета поставщика
- Отображение приоритетных поставщиков в начале списка

#### Избранное
- Добавление поставщиков в избранное для быстрого доступа
- Просмотр списка избранных поставщиков

#### Удаление
- Удаление неактуальных поставщиков из базы
- Запись в историю изменений

### История и аудит

- **Журнал изменений**: запись всех операций с поставщиками
  - Добавление поставщика
  - Редактирование данных (с сохранением старых значений полей)
  - Удаление поставщика
  - Кто выполнил операцию
  - Когда была выполнена операция
  - Какой поставщик был затронут

### Статистика

- Небольшие пометки о работе с поставщиками

## Ограничения

Бот **НЕ включает** следующий функционал:
- ❌ Рассылка сообщений поставщикам из бота
- ❌ Обработка запросов в свободной форме через ИИ
- ❌ Экспорт данных (не требуется на данном этапе)
- ❌ Валидация формата данных (телефон, email)
- ❌ Прикрепление документов и файлов
- ❌ Напоминания о важных датах

## Целевая аудитория

Бот предназначен для использования снабженцами и специалистами отдела закупок.
